# Design Specifications: Agent Command Center

## Design Goal

Translate the provided mockups into a high-fidelity design system and
interactive components for the Command Center.

## Deliverables Needed

1. **Design System:** Colors (Purple/Violet accent), Typography (Inter/Sans),
   Component Library (Buttons, Cards, Modals).
2. **Interactive States:** Hover effects for agent cards, drag-and-drop
   animations for Kanban cards, sidebar transitions.
3. **Mobile/Tablet Layout:** Responsive variations for the grid and board views.
4. **Dark Mode:** A clean, dark theme version of the dashboard.

## Specific View Requirements

- **Agent Cards:** Circular avatars with status rings, skill pills at bottom.
- **Kanban Card:** Header with drag handle, priority badges, metadata footer
  (comments, attachments).
- **Knowledge Card:** Source-coded badges (e.g., Purple for Research, Pink for
  Manual).
