### 3. **Agent Directory Detail View**
**Purpose:** Focused agent management interface

**Key Elements:**
- Large agent avatars (64x64px)

- Search functionality

### 4. **Knowledge Base Interface**

**Key Elements:**

- Cross-references between related knowledge
- Knowledge source attribution

**Knowledge Card Design:**
- Agent avatar + timestamp

- Title and preview text
- Related tags

## 🎨 Design System Specifications

### Color Palette
```
Primary Dark:     #1a1a1a (Background)
Secondary Dark:   #2d2d2d (Cards/Panels)
Accent Purple:    #8b5cf6 (Primary actions)
Accent Blue:      #3b82f6 (Secondary actions)
Success Green:    #10b981 (Status indicators)
Warning Orange:   #f59e0b (Alerts)
Danger Red:       #ef4444 (Errors)
Text Primary:     #ffffff (Headlines)
Text Secondary:   #94a3b8 (Body text)
```

### Typography Scale
```
H1: Inter Bold 32px     (Page titles)
H2: Inter Bold 24px     (Section headers)
H3: Inter SemiBold 18px (Card titles)
Body: Inter Regular 14px (Main text)
Small: Inter Regular 12px (Metadata)
Code: JetBrains Mono 14px (Technical content)
```

### Component Standards
```
Border Radius:    8px (Cards), 4px (Buttons)
Shadows:         0 4px 6px -1px rgba(0,0,0,0.1)
Spacing Grid:    8px base unit
Card Padding:    24px
Button Height:   40px (Primary), 32px (Secondary)
Input Height:    40px
Avatar Sizes:    24px (Small), 32px (Medium), 48px (Large)
```
